Aide mémoire des sélecteurs CSS déja vus

  • * sélectionne TOUS les éléments de la page HTML
  • .ma-jolie-nav sélectionne tous les éléments ayant le nom de class "ma-jolie-nav"
  • #nom-unique-id sélectionne l'élément ayant le nom d'id "mon-unique-id"
  • div, article, aside sélectionne toutes les balises <div>, <article> ET <aside>
  • nav a sélectionne tous les liens de la nav et active une animation lors du survol de la souris
