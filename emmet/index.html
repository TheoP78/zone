HTML

Ajouter un élèment enfant: ">"

li>p
ul>li>p

Résultat :

ul>li>p+a

Résultat :

Multiplier un élément : "*"

ul>li*3>p+img

Résultat :

Ajouter une classe : "."

ul>.wrapper
.container

Résultat :

Grouper des éléments multiples : "()"

ul>(header>ul>li*2>a)+footer>p
ul>(li>p+img+a)+li*2>p+img

Résultat :

Insérer du text dans un élément : "{}"

a{click me}
a{click me}+a{here}+{to continue}

Résultat :

Ajouter une numérotation : "$"

ul>li.item$5

Résultat :

Commencer la numérotation au nombre voulu : "$@x

ul>li.item$@3*5

Résultat :

Numérotation dans l'ordre décroissant: "$@-"

ul>li.item$@-{pizza $@-}*5

Résultat :

Remonter d'un élément enfant : "^"

div>p>em{hello}^a

Résultat :

Ajouter un attribut : "[attr_name="valeur"]

img[src="https://picsum.photos/seed/picsum/200/300"]

En bref : "^"

div.list>ul>(li.item$>p{paragraph $}+img[src="http://placecage.com/300/300"])+li.item$@2*4>p{paragraph $@2}+img[src="http://placesheen.com/$@1$$/$@5$$"]+a^^^footer>p{Place cage vrs placesheen}

Résultat :

Le premier regroupement contient un <li> qui a 2 enfants : <p> qui contiennent du texte et une <img> qui à un attribut scr.

Le regroupement suivant contient les <li> 2 à 5, en multipliant les listes les noms de classe, contenu de paragraphe et valeurs d'attributs s'incrémentent selon des syntaxes différentes. chaque <li> contient aussi une ancre, invariable.

Enfin les 3 ^^^ permettent de remonter les éléments parents pour placer le <footer> au même niveau que la <div> (sortir de <li> puis <ul> puis de <div> pour donner à ce dernier un <footer> "frère").
